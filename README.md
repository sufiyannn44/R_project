The House Rent Prediction project is a machine learning-based application developed using the R programming language. The main objective of this project is to predict the rent of a house based on different features such as area, number of bedrooms (BHK), number of bathrooms, location, and furnishing status.

In this project, a dataset containing house details is collected and processed using data preprocessing techniques. Missing values are removed and categorical data is converted into suitable formats for analysis. The dataset is then divided into training and testing data to build and evaluate the prediction model.

A Linear Regression algorithm is used to train the model because it helps in finding the relationship between house features and rent prices. After training, the model predicts the rent for new houses based on user input values.

The project also includes graphical visualization to compare actual rent values with predicted rent values. The performance of the model is evaluated using RMSE (Root Mean Square Error), which measures prediction accuracy.
